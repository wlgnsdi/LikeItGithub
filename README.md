## 개선사항
- Architecture 의 부재로 인해서 공동작업, Test를 진행하기가 쉽지 않은 형태로 만들졌습니다. Architecture (MVC, MVP, MVVM) 적인 관점에서 공동작업이 가능하고 Test 가 가능하도록 개선이 되어어야 합니다.
